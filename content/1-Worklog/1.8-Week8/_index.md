### Week 8 Achievements & Learnings:

* **Theoretical Knowledge & Design Skills:**

  * **System Design:** Studied and used Draw.io effectively to create and present AWS infrastructure architecture diagrams in a professional and visual manner.
  * **Identity Management:** Understood the principles of Amazon IAM for resource access control and Amazon Cognito for application user authentication.
  * **Centralized Governance & Encryption:** Learned how to manage multi-account environments using AWS Organization combined with AWS Identity Center for Single Sign-On (SSO). Also understood the lifecycle of encryption keys through Amazon KMS.

* **Practical Work (Hands-on Labs):**

  * **Lab 18: Getting Started with AWS Security Hub**

    * Gained a detailed understanding of how AWS Security Hub collects, aggregates, and categorizes security findings from multiple specialized services, such as Amazon GuardDuty for threat detection, Amazon Inspector for vulnerability scanning, and Macie for sensitive data protection, into a single unified dashboard.
    * Successfully configured and enabled the centralized security management service on the AWS account.
    * Analyzed the security compliance posture of the infrastructure by reviewing scores based on different Security Standards.
    * Carefully completed the resource cleanup process after the lab in order to optimize costs and avoid unnecessary resource usage.

* **On-site Office Experience:** On-site work helped make the learning process more focused, while also creating opportunities to quickly discuss with the team whenever technical content or implementation direction needed clarification.

* **Final Project Orientation & Soft Skill Development:**

  * **Final Project Planning:** Conducted team discussions to evaluate practical capabilities and survey cloud infrastructure problems in order to filter possible system ideas. The team agreed to prioritize solutions that are highly applicable, cost-optimized, and closely integrated with information security standards.
  * **Research & Content Planning:** Analyzed “Cloud Journey” stories and real-world case studies from the AWS Study Group community. From these materials, learned how to communicate technical knowledge more effectively and prepared a list of ideas for the team’s own blog channel.
  * **Accumulated Experience:** Becoming familiar with AWS Security Hub created a strong foundation for future security architecture planning. At the same time, this week helped strengthen teamwork mindset, digital content planning skills, and the ability to define a clearer task allocation roadmap for the final project.
